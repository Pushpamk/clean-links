Clean Links Extension Test Page

Testing Instructions:

  1. Install the Clean Links extension in your browser
  2. Copy any of the URLs below by selecting the text and pressing Ctrl+C (Cmd+C on Mac)
  3. You should see a "Link cleaned!" notification appear
  4. Paste the URL somewhere to verify query parameters were removed
